Ketika saya melihat bahwa tempat tidur saya di asrama berantakan, saya menduga ada orang lain yang menggunakan kebutuhan sehari-hari saya. Saya merasa bahwa kewarasan pribadi saya telah terpengaruh.
Nama keluarga saudara laki-laki saya berbeda dari keluarga kami. Dia sering menyeringai di rumah dan memprovokasi orang lain untuk kesenangannya. Hal ini membuat saya marah.
Pada bulan September 1984, saya dipaksa untuk tinggal dengan seseorang yang tidak saya sukai. Minggu pertama kami berselisih dan percakapan kami seperti debat. Atmosfernya sangat buruk. Ketika dia mengetahui bahwa saya telah mendapatkan 95 tanda dalam sebuah tes, dia mengatakan kepada saya sebuah kegagalan, Anda kehilangan 5 tanda. Saya sangat marah, bukan karena nilai saya lebih rendah dari dia, tapi karena saya tidak bisa mentolerir harga dirinya. Saya tidak menunjukkan ketidakpuasan saya dan terus melakukan pekerjaan saya.
Dua tahun yang lalu seseorang mengundang saya untuk menjadi tutor putrinya. Putri bertanya kepada saya beberapa pertanyaan dalam matematika, jadi saya mengajarinya. Namun dia tidak mendengarkan saya dan ini membuat saya merasa tidak bahagia. Tahun kedua itu sama. Ketika saya masuk universitas, orang tua gadis tersebut menyarankan agar saya dipekerjakan sebagai tutor daghners mereka. Mereka mengatakan kepada saya bahwa di universitas tidak ada pekerjaan rumah dan saya akan memiliki banyak waktu sehingga mereka membuat tabel waktu untuk saya yang mengharuskan saya menjadi tutor lima hari dalam seminggu. Mereka tidak menghormati saya dan lagi pula saya memiliki anak lain untuk diajar.
Saya sedang marah
