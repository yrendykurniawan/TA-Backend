Seorang teman mengatakan kepada saya bahwa dia telah menderita cinta dan akademisi. Saya berbagi kesedihannya.
Saya mendapat nilai tes matematika saya dan tidak percaya itu benar. Dulu saya paling tidak mendapat 80% tanda. Apakah saya semakin lemah dalam penelitian? Hal ini membuat saya merasa bahwa saya seharusnya tidak berharap terlalu banyak untuk pergi ke universitas.
Hasil pemeriksaan saya tidak memuaskan meski saya sudah mencoba yang terbaik. Lalu aku mulai bertanya-tanya tentang kemampuanku untuk topik utama. Namun saya tidak tahu bagaimana memilih subjek utama saya jika saya harus mengubahnya.
Tadi malam saya tidak melakukannya dengan baik di salah satu mata pelajaran. Sebelum hasilnya diumumkan, saya khawatir bahwa saya akan gagal. Akhirnya saya mendapat D dan karena hasil pelajaran lainnya tidak semua bagus saya merasa sedikit sedih.
Suatu malam ayahku tiba-tiba menderita serangan jantung yang menyebabkan kesulitan bernafas.
Selama sesi percobaan Fisika saya tidak mengerti isi eksperimen dan tidak tahu bagaimana melakukannya.
Ketika saya di usia 6, ayah saya meninggal. Saya masih sangat muda saat itu dan tidak tahu apa yang telah terjadi. Tangisan kerabat dan air mataku membuatku sedih.
Masalah ini membuatku khawatir, bukannya membuatku sedih. Pada pertengahan bulan September saya pergi ke Guanggho sendiri. Aku sedang mencari akomodasi di malam hari dan setelah berjalan 2 atau 3 mil aku masih tidak bisa menemukan apa-apa. Apa yang dapat saya lakukan? Saya tidak khawatir tidur di jalanan tapi karena beberapa masalah prosedural. Akhirnya aku menemukan tempat untuk tidur jam 8.
Saya telah mengambil tanggung jawab untuk melakukan sesuatu dan saya telah mempersiapkannya untuk itu. Namun saya gagal karena rasa takut saya. Setelah tiga usaha saya masih belum bisa beradaptasi dengan atmosfer dan gagal seperti sebelumnya. Saya merasa tidak kompeten dan merasa bahwa yang lain akan berpikir bahwa saya tidak siap untuk itu.
